*1492. The kth Factor of n

Solution :

class Solution {
    public int kthFactor(int n, int k) {
        // we can generate 2 factors in sqrt(n) operations rather than n ops
        for(int i = 1; i < Math.sqrt(n); ++i) 
            if(n % i== 0 && --k == 0) 
                return i;                       
        for(int i = (int) Math.sqrt(n); i >= 1; --i) 
            if(n % (n/i) == 0 && --k == 0) 
                return n/i;          
        return -1;
    }
}
